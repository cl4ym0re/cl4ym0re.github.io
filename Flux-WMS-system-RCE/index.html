Flux WMS system RCE

cl4ym0re
2023-03-20

Introduction

The FLUX WMS system is a highly productized warehouse management system independently developed by Fuller and honed for 12 years, integrating the best practices of hundreds of enterprises and customers in various industries.A file leak exists in the Flux WMS system that leads to remote code execution

Proof of concept

A vulnerable system might be looked like this:

now we access the /monitor/actuator/heapdump

Since the system uses the Apache shiro framework by default, we can use some heapdump analysis tools like JDumpSpider to get the shiro key from memory

Now we can use this key to implement a shiro deserialization attack.

Exploitation

Use some tools like shiro_attack from github.Specify the key and use AES-GCM mode.
